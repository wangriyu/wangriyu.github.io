流程：
当一个请求 request 进来的时候，server 会依次根据 ServeMux.m 中的 string（路由表达式）来一个一个匹配，
如果找到了可以匹配的 muxEntry，就取出 muxEntry.h，这是个 handler，
调用 handler 中的 ServeHTTP（ResponseWriter, *Request）来组装 Response，并返回。
